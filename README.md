# Curzr.

[curzr.com](https://curzr.com)

Curzr. is a website which provides tools and resources that help you to create a fascinating, interesting, and animated cursor by just a ‘copy and paste’.

## Project setup
```
npm install
npm run serve
```

```
npm run build
npm run lint
```
